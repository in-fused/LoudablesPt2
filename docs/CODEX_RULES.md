# CODEX RULES

## Delivery Rules

- Keep changes additive.
- Respect Phase boundaries.
- Do not replace approved structure.
- Keep dependencies minimal.
- Preserve UI/logic/content separation.

## Phase 1 Build Rules

- Scaffold only.
- Mobile-first app shell only.
- Placeholder scene + bottom drawer only.
- Starter JSON content only.
- PWA baseline only.

## Prohibited in Phase 1

- Backend integration
- Authentication
- Analytics
- Speech recognition
- Full lesson engine
- Large feature expansions
