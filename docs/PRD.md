# PRD (Phase 1)

## Product

Loudables is a mobile-first, conversation-first Spanish learning PWA for real family-life communication.

## Primary Device Target

- iPhone Safari
- iPhone installed home-screen PWA

## Phase 1 Goals

- Set up repository scaffold and stable structure.
- Ship portrait-first app shell that feels mobile-native in the browser.
- Provide scene placeholder and bottom drawer placeholder.
- Include starter JSON content for Module 1 (Family House Arrival).
- Enable installable PWA baseline and service worker registration.
- Add minimal local progress persistence scaffold.

## Out of Scope (Phase 1)

- Full lesson system
- Speech recognition
- Authentication
- Analytics
- Backend services
- Drill-heavy mechanics
- Desktop-first UI patterns
